# 🔢 Digit Identification using Machine Learning

A machine learning project that classifies handwritten digits (0–9) using the Scikit-learn Digits dataset. The project demonstrates the complete workflow of loading data, preprocessing, training a model, evaluating performance, and making predictions.

---

## 📌 Project Overview

This project uses the built-in **Scikit-learn Digits Dataset**, which contains images of handwritten digits represented as 8×8 grayscale pixel values.

The objective is to train a machine learning model capable of accurately recognizing handwritten digits.

---

## 🚀 Features

- Load and explore the Scikit-learn Digits dataset
- Visualize handwritten digit images
- Data preprocessing
- Train a machine learning classification model
- Evaluate model performance
- Predict handwritten digits
- Simple and beginner-friendly implementation

## 📊 Dataset

The project uses the **Scikit-learn Digits Dataset**.

Dataset Information:

- Samples: 1797
- Classes: 10 (Digits 0–9)
- Image Size: 8 × 8 pixels
- Features: 64 pixel intensity values
